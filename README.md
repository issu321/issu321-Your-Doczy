<h2 align="center">⚡ Quick Start</h2>

<div align="center">

### Prerequisites

<p>
  <img src="https://img.shields.io/badge/Python-3.11+-3776AB?style=for-the-badge&logo=python&logoColor=white" />
  <img src="https://img.shields.io/badge/pip-21.0+-3776AB?style=for-the-badge&logo=pypi&logoColor=white" />
  <img src="https://img.shields.io/badge/Git-2.30+-F05032?style=for-the-badge&logo=git&logoColor=white" />

### Step 1: Clone the Repository

```bash
git clone https://github.com/issu321/issu321-Your-Doczy.git
cd issu321-Your-Doczy
```

### Step 2: Create Virtual Environment

```bash
# Windows
python -m venv venv
venv\Scripts\activate

# macOS / Linux
python3 -m venv venv
source venv/bin/activate
```

### Step 3: Install Dependencies

```bash
pip install -r requirements.txt
```

### Step 4: Run the Application

```bash
python run.py
```

### Step 5: Open in Browser

Navigate to `http://localhost:5000` and start converting!

<h2 align="center">🚀 Deployment Guide</h2>

<div align="center">

### Option 1: Deploy to Render (Recommended)

<pre>
    ┌─────────────┐     ┌─────────────┐     ┌─────────────┐
    │   GitHub    │────▶│   Render    │────▶│   Live URL  │
    │   Repo      │     │   Platform  │     │   HTTPS     │
    └─────────────┘     └─────────────┘     └─────────────┘

    Steps:
    1. Push code to GitHub
    2. Create new Web Service on Render
    3. Connect GitHub repo
    4. Set build command: pip install -r requirements.txt
    5. Set start command: gunicorn run:app
    6. Deploy!
</pre>

### Option 2: Deploy to Heroku

```bash
# Install Heroku CLI
# Login
heroku login

# Create app
heroku create your-doczy-app

# Add buildpacks
heroku buildpacks:add heroku/python

# Deploy
git push heroku main

# Open app
heroku open
```

### Option 3: Deploy to PythonAnywhere

```bash
# 1. Upload files via Git or SFTP
# 2. Create virtual environment
mkvirtualenv --python=/usr/bin/python3.11 doczy-env

# 3. Install dependencies
pip install -r requirements.txt

# 4. Configure WSGI file
# Edit /var/www/username_pythonanywhere_com_wsgi.py
# Point to your run.py

# 5. Reload web app
```

### Option 4: Docker Deployment

```dockerfile
# Dockerfile
FROM python:3.11-slim

WORKDIR /app
COPY requirements.txt .
RUN pip install --no-cache-dir -r requirements.txt

COPY . .
EXPOSE 5000

CMD ["gunicorn", "-b", "0.0.0.0:5000", "run:app"]
```

```bash
# Build and run
docker build -t your-doczy .
docker run -p 5000:5000 your-doczy
```
